# Demo Script

Use this as a 60-90 second screen recording script.

## Hook

```text
I built a Discord server where simulated AI research agents debate with you like a chaotic study group.
```

## Short Version

```text
This is Karpathy's Basement.

It is a Discord multi-agent simulator where each research persona has its own Discord bot account.

So instead of one chatbot pretending to be a panel, the room behaves more like a group chat.

If I ask Ilya whether scaling is dead, he answers from a scaling and generalization angle.

Then Yann can jump in and push back with world models.

Demis might reframe the whole thing around planning and evaluation.

Karpathy usually brings it back to data, training dynamics, and the messy reality of building models.

The hidden part is a Gemini-powered room director.

It decides who should speak next, what they are responding to, and what topic they have to stay locked on.

There is also SQLite memory, tone settings, typing delays, contextual pings, and a secret scanner before commits.

The goal is simple: make studying AI research feel less like reading static PDFs and more like sitting inside an opinionated Discord study room.
```

## Demo Flow

1. Show the Discord server with the bots online.
2. Ask:

```text
ilya is scaling dead?
```

3. Let one agent answer first.
4. Let another agent react.
5. Ask:

```text
demis?
```

6. Show that Demis responds to the previous point instead of acting unaware.
7. Show a tone command:

```text
!tone strict
```

8. Ask:

```text
yann i think llms can learn world models from text alone
```

9. Close on the GitHub repo.

## Closing Line

```text
It is not pretending these are the real researchers. It is an educational simulator for making AI research feel more alive.
```
